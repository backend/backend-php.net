    import(array $reference) : \Backend\Core\Utilities\Config

    The import references are defined as follows: imports: - { [type]: [value] }

    Where type can be either

    file

    or

    resource

    . The

    resource

    type follows the same semantics as that of the getNamed method and takes precedent over the file directive.
